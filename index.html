Pizza Pie Chart 是建立在来自 Adobe 的 Snap SVG 结构基础上的圆饼图（pie chart），透过 HTML 和 CSS 而非 JavaScript

url：http://zurb.com/playground/pizza-pie-charts

github：https://github.com/zurb/pizza

browser：官方未说明

resume：主要用来生成饼状图的库

设置每部分数据的颜色的方式有2种，一种是设置CSS样式即可，参考本文件头部的CSS样式

第二种是设置参数 //data_color: ["#CCFF99", "#FF6666","#336699","#009966","pink"],//也可设置每部分数据的颜色，如果没设置，默认为读取样式的值
